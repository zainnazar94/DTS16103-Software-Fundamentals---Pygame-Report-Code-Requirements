To access the 'Pong game' i have created: --------------Please Read Below------------------

install python versions 3.7 whether this be through Anaconda - Spyder, Pycharm or other. To download Anaconda (Python 3.7/3.8) click this link here https://www.anaconda.com/products/individual
After installing Python, install pygame through the anaconda or Python command prompt and type 'pip install Pygame'.
the code can be run through pycharm or python itself. To know if Pygame was successfully installed, type 'import pygame' into your console on Spyder or Python. If a message along th lines of 'Hello from the Pygame community' is displayed, it has been successful.
copy and paste the code from github repo into the python console/text window and press enter to begin the code 5.The game should then load on a new window.
